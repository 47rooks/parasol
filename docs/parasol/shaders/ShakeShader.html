class ShakeShader

package parasol.shaders

extends FlxGraphicsShader

@:directlyUsed@:build(openfl.utils._internal.ShaderMacro.build())@:autoBuild(openfl.utils._internal.ShaderMacro.build())

The ShakeShader shakes the image by manipulating the vertexes. Blur may be added and the shake intensity and duration are configurable.

Constructor

@:glVertexHeader("\r\n uniform bool shakeOn;\r\n uniform vec2 shakeOffset;\r\n ")@:glVertexBody("\r\n if (shakeOn) {\r\n gl_Position.xy += shakeOffset;\r\n }\r\n ")@:glFragmentSource("\r\n #pragma header\r\n\r\n uniform mat3 blur_kernel; // Simple 9 element blur kernel.\r\n uniform bool shakeOn; // True if shaking is on, false otherwise.\r\n uniform bool blurOn; // True if using blur, false otherwise.\r\n uniform float blurOffset; // offset distance to sample about the current texel when blur is on.\r\n\r\n void main() {\r\n vec4 color = vec4(0.0);\r\n vec3 sample[9];\r\n float offset = blurOffset / openfl_TextureSize.x;\r\n mat3 offsets_x = mat3(-offset, 0.0, offset, -offset, 0.0, offset, -offset, 0.0, offset);\r\n mat3 offsets_y = mat3(offset, offset, offset, 0.0, 0.0, 0.0, -offset, -offset, -offset);\r\n \r\n if (shakeOn && blurOn) {\r\n float x = openfl_TextureCoordv.x;\r\n float y = openfl_TextureCoordv.y;\r\n \r\n // Sample texture offsets using the convolution matrix\r\n for (int i=0; i<3; i++) {\r\n for (int j=0; j<3; j++) {\r\n sample[i*3 + j] = flixel_texture2D(bitmap, vec2(x + offsets_x[i][j], y + offsets_y[i][j])).rgb;\r\n }\r\n }\r\n // Apply the blur kernel to the samples\r\n for (int i=0; i<3; i++) {\r\n for (int j=0; j<3; j++) {\r\n color += vec4(sample[i*3 + j] * blur_kernel[i][j], 0.0);\r\n }\r\n }\r\n color.a = 1.0;\r\n } else {\r\n color = flixel_texture2D(bitmap, openfl_TextureCoordv);\r\n }\r\n gl_FragColor = color;\r\n }\r\n ")new()

@:value({ blurOffset : 1.0, blur : false, duration : 0.5, intensity : 0.05 })shake(intensity:Float = 0.05, duration:Float = 0.5, blur:Bool = false, blurOffset:Float = 1.0):Void

Start shake effect with the specified intensity, duration and optional blur.

Parameters:

intensity

maximum proportion (0.0 to 1.0) of screen size to shake in either direction

duration

the time in seconds the shake will last

blur

if true blur while shaking, else do not blur

blurOffset

distance to sample around the current pixel when blurring in number of pixels

update(elapsed:Float):Void

Update the shader variables.

The primary reason for this function is to update the remaining time to shake and randomize the intensity.

Parameters:

elapsed

elapsed time since last update call
