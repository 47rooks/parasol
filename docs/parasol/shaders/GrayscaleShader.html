class GrayscaleShader

package parasol.shaders

extends FlxGraphicsShader

@:directlyUsed@:build(openfl.utils._internal.ShaderMacro.build())@:autoBuild(openfl.utils._internal.ShaderMacro.build())

The GrayscaleShader converts the pixel color value to a grayscale value resulting in a grayscale image.

For the grayscale conversion constants see https://en.wikipedia.org/wiki/Grayscale#Luma_coding_in_video_systems.
