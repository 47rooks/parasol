BlurShader

BlurShader produces a blur shader using a Gaussian kernel.

CombiningShader

A CombiningShader combines two textures into one output. They are the sourceBitmap and the bitmap bitmaps. This shader is expected to be used in a openfl.filters.BitmapFilter subclass, which is expected to set the sourceBitmap.input field before returning this shader to the filter driver code. See parasol.filters.BloomFilter for example usage.

GrayscaleShader

The GrayscaleShader converts the pixel color value to a grayscale value resulting in a grayscale image.

PixelationShader

SineRippleShader

The SineRippleShader creates a ripple effect using a modified sinusoid. Coeffecients of the various parts of the equation permit modification of the frequency of the ripples, the speed at which they radiate from the center, the center or point of impact that starts the ripple, the amplitude at the point of impact, the damping of the ripple effect at greater distances from center.

ThresholdShader

The ThresholdShader converts the pixel color value to a grayscale value and thresholds it at a particular value. The threshold value may be updated by calling the update() function.
