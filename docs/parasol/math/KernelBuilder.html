class KernelBuilder

package parasol.math

@:directlyUsed

The KernelBuilder class provides a builder for kernels with a fluent chaining API. For GAUSSIAN kernel these are the defaults:

 size - no default - must be specified
 clamp - false
 mean - 0.0
 sigma - 0.0
 range - 6
 normalize - true

Refer to the individual APIs for the details of specifying each parameter.

Example:

  var kernel = new KernelBuilder(KernelType.GAUSSIAN)
                  .size(5).stddev(2.0).normalize(false).emit();

will produce a kernel of 5 values from a normal distribution with mean of 0.0 and standard deviation of 2.0 not normalized.

clamp():KernelBuilder

Map the specified sigmaRange of the distribution onto the kernel size. For example if clamp() is called and sigmaRange is 6 then 6 sigma range of the PDF is corresponds to the k elements of the kernel. If sigmaRange() is not called a default range of 6.0 sigma is used. If not called independent mean and kernel size are used.
