# Developer Readme

## Setting up the development environment

```
git clone github.com/parasol
cd parasol
haxelib newrepo
haxelib install libs.hxml
```

## Building the tests

Use VSCode with the vshaxe plugin.

or to build from the command line:

```
lime build hl
```

## Build the haxelib

Go to a command shell or Powershell
```
release.bat
```
